== Description ==

This plugin allows you to specify custom titles for categories. This is especially useful when you are using category pages for content and want to optimize the title.

This plugin is by the developers at <a href="http://www.noannualfeecreditcard.com.au">No Annual Fee Credit Card</a>

== Installation ==

How to install plugin:

e.g.

1. Upload `custom-cat-titles` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
